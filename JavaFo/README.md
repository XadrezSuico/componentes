## JaVaFo
O JaVaFo (que pode ser chamado também de javafo ou JAVAFO dependendo da situação, mas nunca diferente destas três opções por causa do motivo deste nome, que são as iniciais dos criadores) é um software de emparceiramento de Sistema Suíço desenvolvido em Java.

No XadrezSuíço atualmente é utilizado no projeto "Emparceirador" onde a partir da versão 0.0.2 utiliza dele para efetuar o emparceiramento do Sistema Suíço.

Conheça mais o projeto do JaVaFo em: [http://www.rrweb.org/javafo/](http://www.rrweb.org/javafo/)
